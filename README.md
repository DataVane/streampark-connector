## 🚀 什么是streamx-connector
&nbsp;&nbsp;&nbsp;&nbsp; streamx-connector致力于解决flink|spark sql connector稀缺问题,让flink|spark 连接数据更简单容易,streamx-connector这个项目刚起来, 如果你有好用的connector,不妨贡献出来,一块完善,共同建设,你不来贡献等谁来贡献,
